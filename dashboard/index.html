📡 No Dropout — Baseline
No intervention. Every share is passed through as-is. This is what happens today on social platforms — content spreads freely across all community boundaries. False content reaches 54 nodes on average.
🛡 Algorithm 2 — InfoShield Active
Selective suppression. At each spread step, InfoShield solves a linear programme to find the minimum dropout rates that slow false content while keeping true content flowing. Cross-community edges (where false content travels) get a low pass probability. Result: false content drops to 38 nodes — a 30% reduction.
α (alpha) controls how much true content must be preserved. Higher α = stronger protection for true content.

d* Dropout Matrix — What InfoShield Does
Each cell (row u → col v) shows the probability that content is allowed to pass from community u to community v. Green = 1.0 (keep all), Red = 0.0 (suppress all). The algorithm keeps diagonal cells high (within-community flow stays intact) and suppresses off-diagonal cells (cross-community false content is blocked).
